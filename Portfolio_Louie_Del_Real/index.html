This Web Portfolio includes examples of my work in Mr. Lanus' Web Design class. It also includes my reflections (below) on what I learned in each unit of the course.

Unit 1: HTML

  • What do you think are the most important vocabulary terms (or key concepts) in this unit? What do you think that they mean?

    Body,doctype, main, h1,h aref=””, and many more. They are used to create links, different fonts, add text, create the skeleton, many and many different possibilities, and many other different tags, those are just a few that I listed.

  • What do you think are the most important skills that you learned in this unit of study?

    I learned how colors play an important part in the way a website can turn out, and how to use tags like p, h1, main, and many more.

  • In what ways have you gotten better at this kind of work?

    I've learned about how to use tags, and how they work and I know how to start a website from scratch.

  • In what ways do you think you need to improve?

    I need to be more patient when it comes to web design, because this class requires a lot of patience.

  • What problems did you encounter while you were working on this piece? How did you solve them?

    It's not really a problem it's more of a bad habit, I would forgot to add the end dash on almost every end tag.

  • What was especially satisfying to you about either the process or the finished product?

    I really like the finished product because it looks like Mr. Lanus’, and that's the look I was going for.

  • If you were the teacher, what comments would you make about this piece?

    Nothing, I think my piece is beautiful, I would say “this piece is beautiful.”
